# Bone generator
> Yeoman generator for koa and vueJs

## Usage
Install yo,  gulp
```
npm install -g yo gulp
```
Make a new directory, and cd into it
```
mkdir my-new-project && cd 
```
install bone
```
npm install -g generator-bone 
```
use bone
```
yo bone
```

## Generators
* [bone:update](#update)
### update
```
yo bone:update
```
